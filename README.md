# AI Security System 

A real-time human detection system using YOLO and OpenCV with GUI.

## Features
- Live webcam feed
- Human detection (YOLOv8)
- Image capture with timestamps
- Logging system
- Voice alerts
- Telegram alerts (optional)

## Tech Stack
- Python
- OpenCV
- Ultralytics YOLO
- Tkinter

## Setup

git clone <your-repo-link>
cd motion_detection
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python gui.py

## Output
- Images → /captures
- Logs → log.txt
